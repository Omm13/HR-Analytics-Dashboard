# HR Analytics Dashboard

## Project Overview
Interactive HR Analytics Dashboard built in Microsoft Excel using Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

## Objectives
- Analyze workforce demographics
- Track employee performance
- Monitor attrition trends
- Evaluate training investment

## Key Metrics
- Workforce Size
- Active Workforce
- Attrition Rate
- Average Employee Age
- Employee Satisfaction Score
- Total Training Cost

## Dashboard Features
- Department Analysis
- Age Group Distribution
- Employee Status Breakdown
- Performance Score Analysis
- Attrition by Department
- Training Budget Allocation

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
